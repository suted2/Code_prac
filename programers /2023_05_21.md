### 1. 최솟값 만들기

[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/12941)


```python
def solution(A,B):
    answer = 0
    
    A = sorted(A)
    B = sorted(B, reverse = True)
    ## 곱이 최소이기 위해서 한쪽의 제일 큰수 * 다른쪽 제일 작은 수 방식으로 진행해 나가야한다. 
    
    for i,j in zip(A,B):
        answer += i * j 
    
    return answer

```
