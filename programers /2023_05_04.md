### 1. 문자열 내림차순으로 배치하기
---
```python
def solution(s):
    answer = ''
    lower = [] 
    upper = [] 
    
    for i in s:
        if i.isupper():
            upper.append(i)
        else:
            lower.append(i)
    
    lower.sort(reverse = True)
    upper.sort(reverse = True) 
    
    tmp = lower + upper 
    
    return ''.join(tmp)


```
