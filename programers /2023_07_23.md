### 덧칠하기 
[링크](https://school.programmers.co.kr/learn/courses/30/lessons/161989)



```python
def solution(n, m, section):
    answer = 1 # 칠하는 횟수
    paint = section[0] # 덧칠 시작점
    for i in range(1, len(section)):
        if section[i] - paint >= m:
            answer += 1
            paint = section[i]
            
    return answer
```
