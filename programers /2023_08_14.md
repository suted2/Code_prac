## 가장 큰 수 

[문제 링크](https://school.programmers.co.kr/learn/courses/30/lessons/42746)


```python


def solution(numbers):
    answer = ''
    temp = []
    for i in numbers:
        temp.append(str(i))

    temp.sort(key = lambda x: x*3, reverse = True)
    
    
    
    return ''.join(temp)


```
